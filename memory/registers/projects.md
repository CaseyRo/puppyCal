## puppyICS → PuppyCal

- **Product name**: PuppyCal (repo/working name may stay puppyICS or become puppycal).
- **What it is**: Quick-and-dirty solution for managing a puppy’s routine and info—focused on one breed in this case (stabyhoun).
- **What we have**: Current setup and intention captured; implementation to reflect this.
- **Intention / scope**:
  - **Food scheme** — schedule/plan for feeding.
  - **Walking schedule** — when to walk the dog.
  - **Random information** — breed-specific info (e.g. stabyhoun facts, tips, links).
- **Tone**: Pragmatic, minimal; not a full product.
- **js-webserver-ics (planned)**:
  - Frontend: JS + simple webserver; Tailwind, form-based, mobile-first; ICS generated in browser.
  - NFR: RFC 5545–compliant ICS (Outlook, Gmail, Apple Mail); all options via URL params and UI (DOB, plan months, start date, birthday reminders, optional dog content).
